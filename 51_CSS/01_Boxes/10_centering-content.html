If you want to center a box on the page, or center it inside the element it sits in, you can set the left-margin and right-margin to auto.

In order to center a box on the page, you need to specify the width for the box (otherwise it will take up the full width of the page).
For this to work in older browsers, the element that the box sits inside should have a text-align: center property-value. Then for each child element, you should set the desired text-align property:value.
