CSS3 introduces the ability to create rounded corners on any box, using a property called border-radius. That balue indicates the size of the radius in pixels.
Older browsers that do not support this property will display square corners.

The -miz-border-radius and -webkit-border-radius properties are not in the the CSS specification. However, they are used in some versions of Chrome, Firefox, and Safari to offer early support for this style.

You can specify individual values for each corner of a bix using: border-top-right-radius, border-bottom-right-radius, etc.
You can also specify corner radius using shorthand -> border-radius: 5px 10px 7px 12px;
