Here are some properties that can be used to style submit buttons. This example builds on the one in the previous page, and the submit button inherits the styles set for the input element on the last page.

color is used to change the color of the text on the button.
text-shadow can give a 3D look to the text in browsers that support this property.
border-bottom has been used to make the bottom border of the buttons slightly thicker, which gives it a more 3D feel.
background-color can make the submit button stand out from items around it. (Creating a consistent style for all buttons helps users understand how they should interact with the site.) A gradient background has been added for browsers that support gradients.
The :hover pseudo-class has been used to change the appearance of the button when the user hovers over it.
